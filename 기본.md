### 기본 개념

일반적으로 app store에서 자료를 다운 받는다 도커는 docker hub

다운 받은 program을 도커에서는 image, program을 실행 시켜서 생성되는 process를 container라고 한다.

docker hub -> (pull) -> image -> (run) -> container


```
docker pull httpd
```
